Features

  • Visuelle Darstellung des Tröpfchenmodells der Kernphysik
  • Nuklidkarte mit 4561 Isotopen aller bekannten Elemente
  • Spontanzerfall, Neutroneninduzierter Kernzerfall, Kernfusion auf Knopfdruck

Anmerkungen

Diese Simulation steht unter der GNU GPLv3. Ihr Quellcode ist öffentlich auf Github einsehbar.

Tröpfchen soll für Schüler kostenlos bleiben.
Daher möchte ich freundlich an Unternehmen und Bildungseinrichtungen appellieren, mich finanziell dabei zu unterstützen.

Download

Tröpfchen kann auf der Releaseseite von dem Github-Repository heruntergeladen werden.
Neue Versionen werden dort veröffentlicht und Downloads für Windows, Mac OS X und Linux werden dort bereitgestellt.

Auf Windows wird das .NET-Framework ab Version 3.5, auf Mac OS X und Linux Mono benötigt.
Das .NET-Framework ist auf Windows vorinstalliert, Mono hingegen muss, sofern noch nicht geschehen, heruntergeladen und installiert werden.
Offizielle Homepage vom Mono Project: http://www.mono-project.com/
