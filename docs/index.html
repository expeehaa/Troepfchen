Download

Tröpfchen kann auf der Releaseseite von dem Github-Repository heruntergeladen werden.
Neue Versionen werden dort veröffentlicht und Downloads für Windows, Mac OS X und Linux werden dort bereitgestellt.
Da bisher weder Erinnerungen oder automatische Updates in das Programm integriert wurden, sollten sie ab und zu dort vorbeischauen.

Zum Ausführen wird auf Windows das .NET-Framework ab Version 3.5, auf Mac OS X und Linux Mono benötigt.
Das .NET-Framework ist auf Windows ab XP vorinstalliert, Mono hingegen muss, sofern noch nicht geschehen, heruntergeladen und installiert werden.
Offizielle Homepage vom Mono Project: http://www.mono-project.com/
